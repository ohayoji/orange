Orangeのサロン管理者には「毎日のタスク」と「毎月のタスク」の２種類のタスクがあります

毎日のタスク

来店処理

その日の会計伝票をすべて来店処理して、伝票を完成させましょう。

来店処理のやり方を見る

伝票は、来店処理まで完了してはじめて完成されます。サロン管理者は必ず来店処理を行ってください。

詳しくは伝票ステータスについてをご覧ください。

レジ締め

営業が終わったら、日報の「現金売上」を確認してレジ締めをしましょう。

日報ガイドを見る

毎月のタスク

月報を作成する

１ヶ月間おつかれさまでした。

月が終わったら月報作成画面で経費項目を入力して月報を作成しましょう。

月報作成ガイドを見る
さらに便利な機能

「給与（報酬）承認機能」と「支払い管理機能」を利用することで、給与明細や請求書のやりとりがなくなり、給与振込の作業もカンタンになります。

これらの機能を是非ご利用ください。

便利な機能 給与（報酬）承認 と 支払い管理

給与（報酬）承認

サロンスタッフが自分のアカウントでログインすると「売上明細」を確認することができます。

売上明細ガイドを見る

売上明細の中にある、給与（報酬）明細を承認してもらいましょう。

報酬承認ガイドを見る
支払い管理

サロンスタッフが給与（報酬）を承認すると、サロン管理者の支払い待ちリストに追加されます。

給与（報酬）の支払い（振込）完了時に「支払い済み」に切り替えるだけでOKです。

支払い管理ガイドを見る

予約やお会計に関するデータは「伝票」に保存されます。

伝票に保存される主なデータ

  • 来店日時や顧客名、施述メニューなどの基本データ
  • 施述料金や割引率（技術・商品）
  • メモ
  • 来店回数や支払いタイプ（現金・カード）
  • 担当スタイリストの歩合率（技術・商品）

伝票は予約ごとに１枚づつ発行され、
予約登録　→ ご来店　→ お会計　→ 来店処理
という順序でステータスが変化していきます。
